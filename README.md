# UninstallRedirect
拦截桌面、安全中心以及垃圾清理的应用卸载，重定向至系统默认卸载器
解决InstallerXRevived无法接管应用卸载的方案。
    关于解决在红米k90promax设备上使用InstallerXRevived时无法接管桌面等的应用卸载的解决模块。
    由于厂商魔改，系统桌面、安全服务以及垃圾清理中操作的应用卸载不会调用系统包管理器进行，使得InxLocker模块的拦截卸载功能无法拦截，故本模块应运而生。在使用本方法和本模块后，任何在桌面、设置或是安全中心（手机管家）中的应用卸载入口中操作的卸载，都可由InstallerXRevived或其他第三方卸载器接管。
    模块介绍：名称为smyprl卸载重定向，本模块通过对系统桌面、安全服务以及垃圾清理的Hook，实现了卸载应用时强制使用系统包管理器（com.miui.packageinstaller）卸载，而不是通过自己奇奇怪怪的方式卸载；目前已知可以拦截并重定向的应用卸载入口有：桌面拖动和长按点击卸载，桌面多选拖动和点击卸载，通过任何应用到达的应用信息界面点击卸载，通过手机管家或设置到达的深度清理中的应用卸载，通过点击设置中的手机存储空间中的应用和数据界面等方式进入到的应用占用详情界面的应用卸载。
     本模块完整作用域为安全服务、垃圾清理、系统桌面。
                             完整方法 
    1.由于本模块是将应用卸载重定向至系统包管理（com.miui.packageinstaller）应用，所以想要卸载由InstallerXRevived接管，首先应替换原系统包管理，这里可以使用由酷安@桜_y 制作的用于替换系统安装器的Magisk模块
    https://www.coolapk.com/feed/70894259?s=MzNkMDE3NDIxNzE0NzU3ZzZhNWJhN2Mwega1650
下载上述模块后请认真阅读模块文件内的README.md文档，如果刷入失败并且提示卸载更新（需恢复出厂版本）的内容，应首先尝试使用Mt管理器在/data/app/目录下找到应用包管理所在文件夹，删除里面的文件并重启手机。
    2.安装smyprl卸载重定向模块，选择推荐作用域后，只需将推荐作用域所包含的应用全部重启即可生效，无需重启手机。请注意！应先刷入上述所说的Magisk模块，并保证可以正常使用后再安装并激活smyprl卸载重定向模块。
